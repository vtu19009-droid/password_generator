# Password Generator

A simple, modern Java Swing password generator with a dark UI.

## Features
- One-click password generation using `SecureRandom`
- Adjustable length (4–128 characters)
- Strength indicator (Weak / Good / Strong / Very Strong)
- Copy to clipboard
- No dependencies — pure Java

## Run

```bash
javac PasswordGenerator.java
java PasswordGenerator
```

Requires Java 8+.
